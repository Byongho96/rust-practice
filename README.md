# rust 설치

[rustup](https://rustup.rs/)

# rustc

rust CLI

**rust 버전확인**

```bash
rustc --version
```

**rust 컴파일**

```bash
rustc main.rs
```

# cargo

rust 패키지 관리자

**cargo 버전확인**

```bash
cargo --version
```

**새 프로젝트 생성**

```bash
cargo new hello_cargo
cd hello_cargo
```

**프로젝트 빌드**

```bash
cargo build
```

**프로젝트 빌드+실행**

```bash
cargo run
```

# rust 제거

```bash
rustup self uninstall
```
